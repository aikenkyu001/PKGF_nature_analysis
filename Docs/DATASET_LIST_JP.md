# 自然観測データセット & 論理プリミティブ対応表 (PKGF/MQDB)

本プロジェクトでは、**21個の自然観測データセット**に加え、比較基準となる **1個の合成基準データ（Synthetic Ref.）** を含む、**計22個の幾何学的プロファイル** を解析・モデル化の対象としています。

各データは「幾何学的プロファイル」として解析され、数学的なパラメータ（平均、分散、記憶の強さ、傾向）を持つ論理プリミティブへと凝縮されています。

| ファイル名 | 対象内容 | 概要 |

| `ice_core` | 氷床コア | 南極等の氷から復元された過去数十万年の気候データ（気温・ガス濃度）。 |
| `treering` | 樹木の年輪 | 年輪の幅から読み取る数千年前からの気候変動記録。 |
| `atmospheric_noise`| 大気ノイズ | 雷放電などに由来する高周波の電磁波ノイズ。 |

## 論理プリミティブの定義パラメータ
各データセットは、以下の4つの共通次元を持つ論理プリミティブとして再定義されました。これらは `pkgf_master_pipeline.py` によって抽出されます。

1. **平均 ($\mu$)**: システムの基底レベル。
2. **分散 ($\sigma$)**: システムのエネルギー・変動の激しさ。
3. **Hurst指数 ($H$)**: **記憶の深さ**。0.5（ランダム）〜1.0（強い相関）。
4. **トレンド ($\beta$)**: 長期的な上昇または下降の傾向。

各ファイル（例：`nile`）は、`Data/nile_raw.csv` として実在し、解析結果は `Data/nile_morphic_profile.json` に格納されます。
